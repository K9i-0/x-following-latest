# Privacy Policy / プライバシーポリシー

## English

### Data Collection
X Following Latest does **not** collect, store, or transmit any user data.

### Permissions
This extension only requires permission to run on x.com and twitter.com domains. No other permissions are requested.

### How It Works
The extension operates entirely within your browser by:
- Detecting and hiding the "For You" tab
- Automatically selecting the "Following" tab
- Switching the sort order to "Latest"

All processing is done locally in your browser. No data is sent to external servers.

### Third-Party Services
This extension does not use any third-party analytics, tracking, or data collection services.

### Contact
If you have questions about this privacy policy, please open an issue on the GitHub repository.

## 日本語

### データ収集について
X Following Latest はユーザーデータを**一切**収集、保存、送信しません。

### 権限について
この拡張機能は x.com と twitter.com ドメインでの実行権限のみを必要とします。その他の権限は要求しません。

### 動作の仕組み
この拡張機能はブラウザ内で以下の処理を行います：
- 「おすすめ」タブの検出と非表示
- 「フォロー中」タブの自動選択
- 並べ替え順序を「最新」に切り替え

すべての処理はブラウザ内でローカルに行われます。外部サーバーへのデータ送信は一切ありません。

### サードパーティサービス
この拡張機能は、サードパーティの分析、トラッキング、データ収集サービスを使用していません。

### お問い合わせ
このプライバシーポリシーについてご質問がある場合は、GitHubリポジトリでissueを作成してください。
